## 1. Descripción del Proyecto
Este repositorio contiene la actividad práctica de la Unidad 3, donde exploramos el uso de GitHub Copilot para generar código de Python enfocado en Machine Learning (algoritmo K-NN).

## 2. Desarrollo de la Actividad
Para realizar este trabajo se siguieron los siguientes pasos:

1.  **Configuración del Entorno:** Se creó el repositorio y se clonó en Visual Studio Code. Se instalaron las librerías necesarias (`pandas`, `scikit-learn`, `numpy`).
2.  **Generación de Datos:** Se creó el archivo `products.csv` para entrenar el modelo.
3.  **Programación con Copilot:** Se utilizó IA para generar el script de recomendación.
